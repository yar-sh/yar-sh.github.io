I Wanna Reach The Unity

This is a fangame made in Unity as the final project for my Computer Science grade 12 class. Inspired by many other I Wanna Be The Guy fangames, this game is a medley of several other well-known fangames (ported to Unity, hence the title).

üéÆ controls

  • Left/Right arrows for movement
  • Z for jumping and double jumping OR for chosing an option in the main menu
  • X for shooting
  • C for quick restart
  • Q for self-destruct
  • Tab for showing deaths and time stats
  • F5 for teleporting to stage hub

üïπ dev controls

  • S for quick save
  • Ctrl for teleporting player to the mouse cursor position
  • Page Up for jumping to the next level
  • Page Down for jumping to the previous level
  • I for invincibility
